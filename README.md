# Bishal Fitness Guide

React + Vite version of the fitness plan app.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

Import this folder/repository in Vercel. Vercel should auto-detect Vite.

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
